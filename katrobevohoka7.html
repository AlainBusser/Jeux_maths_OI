But du jeu

il y a 28 graines. On sème les graines une par une, dans le sens de son choix, à partir d'une case non vide, pour réarranger le plateau. Si la dernière graine tombe dans une case non vide, on recommence à semer, dans le même sens, jusqu'à ce que la dernière graine tombe dans une case vide ou dans la cupule verte. Dans ce cas on rejoue. Le but du jeu est de finir avec 28 graines dans la cupule verte.
